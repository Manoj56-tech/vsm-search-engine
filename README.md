# VSM Search Engine 🔍

## Author
Manoj Kumar Shah

## Description
This project implements a simple search engine using the Vector Space Model (VSM).

## Features
- TF-IDF vectorization
- Cosine similarity
- Ranked search results

## How to Run
pip install -r requirements.txt  
python main.py

## Concepts Used
- NLP (Natural Language Processing)
- Vector Space Model (VSM)
- TF-IDF
- Cosine Similarity
